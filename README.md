# 📌 Sobre o Projeto

O arquivo principal deste repositório apresenta o levantamento de requisitos, diagramas e a estruturação inicial de um sistema de software. O foco do material é aplicar metodologias de desenvolvimento e boas práticas de arquitetura.
---

# Conteúdo do Repositório

O foco desta primeira entrega é o levantamento de requisitos e o entendimento profundo do problema através de ferramentas de Design Thinking.

---

# Etapas Desenvolvidas

- **Mapa da Empatia** — Análise detalhada sobre o que o cliente diz, pensa, faz e sente em relação ao problema proposto.

- **Detalhamento de Dores** — Identificação de pontos críticos como dependência de inspeções presenciais, falta de estruturação de dados e falhas na gestão de tempo.

- **Ganhos Esperados** — Mapeamento de benefícios como redução de custos, otimização de tempo e melhor tomada de decisão.

- **Aprofundamento da Ideia** — Consolidação das visões individuais de cada integrante para a solução central.

---

# Tecnologias e Ferramentas Utilizadas

- **Miro** — Utilizado para a confecção colaborativa dos mapas e quadros de ideias.

- **Software Engineering Fundamentals** — Aplicação de conceitos iniciais de análise de sistemas.
